## 📂 Project Contents

This repository contains 5 shell scripts demonstrating Linux and open-source concepts:

### 🔹 Script 1 — System Identity Report

* Displays system information like kernel, user, uptime, and date
* Demonstrates use of variables and command substitution

### 🔹 Script 2 — FOSS Package Inspector

* Checks if Git is installed
* Displays version and description
* Uses `if-else` and `case` statements

### 🔹 Script 3 — Disk and Permission Auditor

* Checks system directories
* Displays permissions and disk usage
* Uses loops and file inspection

### 🔹 Script 4 — Log File Analyzer

* Reads a file line-by-line
* Counts occurrences of a keyword
* Uses loops and conditional logic

### 🔹 Script 5 — Open Source Manifesto Generator

* Takes user input
* Generates a personalized open-source statement
* Saves output to a file

## ⚙️ How to Run the Scripts

### 1. Open terminal (Git Bash)

Navigate to project folder:

```bash
cd ~/git-audit-projects
```

### 2. Make scripts executable

```bash
chmod +x script*.sh
```

### 3. Run scripts

```bash
./script1_system_identity.sh
./script2_package_inspector.sh
./script3_disk_audit.sh
./script4_log_analyzer.sh script1_system_identity.sh
./script5_manifesto.sh
```
